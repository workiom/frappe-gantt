Task Column Resize Feature Test

How to Test Resize Functionality

  • Hover over the right edge of the task column to see the resize handle
  • Click and drag the handle to resize the column
  • Watch the Event Log below to see resize events being triggered
  • task_column_resize fires continuously during drag
  • task_column_resized fires once when you release the mouse
  • Try RTL mode to see the handle move to the left side
